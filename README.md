# El Virus 3D - Frontend

Este es el repositorio del sitio web oficial de **El Virus 3D**, un estudio de desarrollo de videojuegos indie con sede en Uruguay.

## 🚀 Tecnologías
- **React 18** (Vite + TypeScript)
- **Tailwind CSS** (Estilizado y Responsividad)
- **Framer Motion** (Animaciones dinámicas)
- **Lucide React** (Iconografía)
- **Shadcn UI** (Componentes de interfaz)

## 🛠 Instalación y Uso

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/USUARIO/elvirus-3d-frontend.git
   ```
2. Instalar dependencias:
   ```bash
   npm install
   ```
3. Ejecutar en modo desarrollo:
   ```bash
   npm run dev
   ```
4. Construir para producción:
   ```bash
   npm run build
   ```

## 🔐 Panel de Administración
La web cuenta con una sección de administración oculta para gestionar Noticias, Juegos y Eventos sin necesidad de base de datos. Para acceder:
- Haz **5 clicks rápidos** en el título de la sección (Noticias, Juegos, Eventos o Último Lanzamiento).
- Ingresa el PIN de seguridad.
- Genera el código para actualizar el contenido.
